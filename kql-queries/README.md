# KQL Queries – Security Use Cases

Collection of Kusto Query Language queries for Azure security monitoring.  
Queries are organized by use case and based on personal lab experience.

---

## Categories

- **Authentication & Identity** – failed logins, suspicious sign-ins
- **Network** – unusual traffic, firewall hits
- **Resource Changes** – who changed what and when
- **Defender Alerts** – security recommendations and incidents

---

> Queries are written for Log Analytics Workspace and tested in personal lab environment.
